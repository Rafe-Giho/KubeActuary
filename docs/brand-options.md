# Brand Symbol Options

These are project-local SVG candidates for the first KubeActuary symbol. They
are intentionally vector-based so the final choice can be reused in README,
GitHub social previews, docs, stickers, and future UI.

## Option A: Proof Ring

![Proof Ring](../assets/brand/symbol-option-a-proof-ring.svg)

Best fit if KubeActuary should feel Kubernetes-native first. The outer operation
ring references cluster operations, the shield/check expresses proof and gate
approval, and the small ticks suggest measured risk.

Recommendation: strongest default for README and GitHub.

## Option B: Actuary Compass

![Actuary Compass](../assets/brand/symbol-option-b-actuary-compass.svg)

Best fit if the project should emphasize risk navigation and actuarial judgment.
It is more distinctive, but a little less obviously Kubernetes-related.

## Option C: Capsule Gate

![Capsule Gate](../assets/brand/symbol-option-c-capsule-gate.svg)

Best fit if the project should emphasize the `OperationCapsule` artifact itself.
It reads as a gate and evidence graph, but is visually more abstract.

## My Recommendation

Choose **Option A: Proof Ring** for the first public release.

Reason:

- clearly cloud-native;
- communicates evidence and approval quickly;
- works at small sizes;
- does not overclaim AI magic;
- fits the safety-oriented posture of the project.
